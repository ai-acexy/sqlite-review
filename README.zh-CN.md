# sqlite-review

我创建了这个简洁的 SQLite 在线预览与编辑项目，用于在浏览器中加载、查询、修改和导出 SQLite 数据库。

## 环境信息

- 创作代理：Codex
- 使用模型：GPT-5（Codex coding agent runtime）
- 运行环境：Codex desktop

## 在线地址

- 使用地址：[https://sqlite.acexy.cn](https://sqlite.acexy.cn)

## 功能

- 我实现了通过文件选择和拖拽导入 SQLite 文件
- 我实现了在浏览器中新建空数据库
- 我实现了在编辑器中直接执行 SQL
- 我实现了左侧的表浏览与结构查看
- 我将表预览结果移到了主结果区
- 我在编辑器中加入了 SQL 语法着色
- 我加入了可拖拽分隔条，用于调整 SQL 与结果区域比例
- 我增加了导出为 `.sqlite` 文件的能力
- 我增加了带确认的工作区清空操作
- 我为覆盖现有工作区数据的操作加入了二次确认
- 我增加了主题和编辑器内容的本地存储
- 我增加了白天/黑夜主题切换
- 我让页面在桌面端与移动端都能自适应
- 我将 `sql.js` 本地化，并保留了 CDN 优先、失败自动回退的加载方式

## 快速开始

1. 直接在浏览器中打开 `index.html`。
2. 导入 SQLite 文件，或者新建一个空数据库。
3. 在编辑器中输入 SQL，按 `Cmd/Ctrl + R` 执行。
4. 在右侧结果区查看查询结果、结构信息和表预览。

## 项目结构

- `index.html`：页面骨架与 UI 结构
- `bootstrap.js`：轻量运行时启动器
- `style.css`：布局与视觉样式
- `app.js`：SQLite 加载、SQL 执行、预览渲染与交互逻辑
- `public/vendor/sql.js`：本地 `sql.js` 运行时与 WebAssembly 回退资源

## 说明

- 我将本项目实现为纯静态前端方案。
- 无需构建工具，也无需安装依赖即可运行。
- 当前采用 CDN 优先加载，失败时会自动回退到本地 vendor 资源。
